# ShopEase E-Commerce Website

This is a static e-commerce demo website created with HTML, CSS, and JavaScript. It includes:

- Home page with hero slider, category cards, featured products, and features section
- Products listing with category filter and price range slider
- Product details page with gallery, add to cart, and related products
- Cart and checkout workflows using Local Storage
- Search page, categories page, contact form, and order confirmation page

## Run locally

Open any `*.html` file from the `E-Commerce-Website` folder in your browser, or serve the folder with a local web server for full component loading support.
